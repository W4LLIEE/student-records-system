# Student Record System

**Status:** Complete

## Description
A console-based student record system written in C.
The program allows users to create and store student records.

## Features
- Menu-driven interface
- Add new student records
- List all stored student records
- Data persistence using file I/O
- Uses structures and enums for organized data handling

## How to Run
1. Compile the program using a C compiler (e.g. `gcc`)
2. Run the executable in the terminal
3. Follow the on-screen menu instructions
